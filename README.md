# Radar da Vida Escolar 
 
## Descrição 
O Radar da Vida Escolar é um projeto de IA e Dados que investiga dificuldades dos estudantes na organização da rotina escolar. 
 
A proposta é analisar dados coletados com estudantes e transformar os principais padrões encontrados em uma solução de apoio à organização dos estudos. 
 
## Problema investigado 
Muitos estudantes relatam dificuldade para organizar tarefas, provas, revisões e horários de estudo. 
 
O projeto investigou quais fatores aparecem com mais frequência nessa dificuldade, como procrastinação, distrações, falta de cronograma e acúmulo de atividades. 
 
## Público-alvo 
Estudantes da escola que desejam melhorar sua organização escolar e compreender melhor seus hábitos de estudo. 
 
## Dados utilizados 
Foram utilizados dados de uma pesquisa sobre rotina escolar, incluindo informações como: 
 
- série atual; 
- turno de estudo; 
- canais de aviso; 
- horas semanais de estudo; 
- uso de agenda; 
- dificuldade com tarefas; 
- relatos abertos sobre rotina escolar. 
 
## Principais insights 
A análise dos dados indicou temas recorrentes como: 
 
- procrastinação; 
- distrações com celular e redes sociais; 
- falta de cronograma; 
- acúmulo de matérias; 
- dificuldade de foco; 
- dificuldade para conciliar atividades extras e estudos. 
 
## Solução proposta 
A solução proposta é o RadarBot, um assistente de IA criado para responder perguntas sobre os principais padrões encontrados no projeto e sugerir estratégias simples de organização escolar. 
 
## Uso de Inteligência Artificial 
A inteligência artificial foi usada para: 
 
- apoiar a análise de relatos abertos; 
- organizar temas recorrentes; 
- criar uma base de contexto; 
- estruturar o comportamento do assistente; 
- testar perguntas e respostas; 
- apoiar a criação da apresentação final. 
 
## Como acessar o projeto
